leam.sections Installation
--------------------------

To install leam.sections using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``leam.sections`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        leam.sections
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
